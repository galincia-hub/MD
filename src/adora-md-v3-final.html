왜 캐빈 쉐어가 필요한가?

문제: 한 사의 메인 항차라도 그 사의 고객만 살 수 있다면, 다른 사 고객은 그 날짜를 선택 불가 → 가격·일정 매칭 실패 시 캐빈 잔여 발생
해결: 메인 사가 X%를 책임지고, 나머지를 타사가 균등 분할받아 자기 고객에게 판매할 권한 확보 → 모든 사의 고객이 모든 날짜를 선택 가능
효과: 캐빈 활용도 극대화 + 사별 고객층 다양화 + 3사 간 공정성 확보 + 잔여 캐빈 리스크 분산

자율 선택 시: 타사가 안 팔면 메인 사 잔여 책임. 메인 사에게 불리한 구조 → 컨소시엄 신뢰 저하 위험

제X조: 의무 배분 캐빈의 판매 부진 시 책임은 인수한 사가 부담하며, 메인 사는 본인 책임분(80%) 외 추가 부담하지 않는다.

대리점 수수료 민감도 분석 — 봄 시즌 총 수익 변화

BEP (Break-Even Point) 분석 — 판매가 ≥ BEP일 때 흑자

계산식 해설
1인당 원가 = NPD ($109.97 × 박수 × 환율) + 보험료 + 판매가 × (대리점% + 카드% + 마일리지%)
1인당 수익 = 판매가 − 원가
수익률(매출 대비) = 수익 ÷ 판매가
BEP (손익분기 판매가) = (NPD + 보험료) ÷ (1 − 수수료비율)
항차 매출/원가/수익 = 캐빈 타입별 1인 값 × 캐빈 수 × 2명 (DBL 기준)

예시 — 발코니 프리미엄 5박 판매가 ₩2,890,000 (기본 비용 파라미터):
NPD = $109.97 × 5 × ₩1,485 = ₩816,527
수수료 = ₩2,890,000 × 21.5% = ₩621,350
원가 = 816,527 + 20,000 + 621,350 = ₩1,457,877
수익 = 2,890,000 − 1,457,877 = ₩1,432,123 (수익률 49.6%)
BEP = (816,527 + 20,000) ÷ 0.785 = ₩1,065,640 → 현재 판매가가 BEP보다 ₩1,824,360 높음 (안정적 흑자 영역)
